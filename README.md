# Boilerplate based on Node.js with Typescript

## Tech-stack

- Node.js
- Typescript
- Express.js (API)
- Sequelize (Database)
- Celebrate (Request Validation)
- Inversify (Dependency Injection)
- ESLint (Linting)
- Prettier (Formatting)

## Initial install

#### Install the dependency packages
```bash
npm install
```

#### Start the server
```bash
npm start
```

#### Lint the code
```bash
npm run lint
```

#### Lint, fix and format the code
```bash
npm run lint:fix
```
